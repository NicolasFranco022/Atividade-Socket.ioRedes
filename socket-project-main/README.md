## Tecnologias que foram utilizadas
Backend: Node.js, Express, Socket.IO 
Frontend: HTML, CSS, JavaScript |
Gerenciador de Pacotes: npm |
Execução Local: localhost (porta padrão: 3000) |

## Pré-requisitos e comandos
- Node.js instalado
- npm install ou a caso de erro --> npm install socket.io
- npm install ou a caso de erro --> npm install express


### Execução
-Certificar se está dentro da pasta do back, caso não                --> cd back  -> npm start (rodar servidor)
-Após isso abrir outro terminal integrado dentro da pasta do front   --> cd front -> npm run start (rodar react)
